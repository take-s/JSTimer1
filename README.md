Presentation Timer

* これは、学会用などのタイマーです。
* htmlファイル1つと、ベルなどで再生したい音声ファイルで動作します。
* 時間は、講演時間＋質問時間=総時間です。
* 音声ファイルは、講演時間終了時、質問時間終了時、予鈴１、予鈴２の形でファイル名を指定します。
* あとはお好みで適当に書き換えてください。
* リセットを押すと時間がセットされます。

* 初期ですと、講演時間終了(ベル3回)、全体終了(ベル3回)、予鈴ベル1(ベル1回)、予鈴ベル2(ベル2回)です。
